# Установка
Установить node v18.12.1 и выше
Создать файл .env и вставить в него содержимое

> npm i

# Настройка
Можно получить из ключей приложения\
CLIENT_ID\
CLIENT_SECRET

USER_ID - Номер профиля из вкладки: Управление кабинетом

# Запуск

> node index.js

# Остановка
Сочетание клавиш:
> CTRL + C 
Или команда:
> stop
